CSS Stands for the Cascading Style Sheets. CSS is styling language used to provide the powerful appearance for the web page. Most commonly it is used to apply styling over the HTML document.

HTML is used to manage the content of the web page where CSS is use for presenting web page such as fonts, colors, background, borders, text formatting etc. It is easy to learn and understand. Usually, CSS is combined with the mark-up languages HTML or XHTML.
